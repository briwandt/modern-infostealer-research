# The Evolution of Modern Infostealers

Threat-intelligence research examining the evolution of infostealers into identity-centric intrusion platforms targeting SaaS environments, cloud infrastructure, AI tooling, and non-human identities (NHIs).

## Topics Covered
- Session replay and SaaS compromise
- OAuth abuse and cloud pivoting
- AI-enabled phishing and social engineering
- LLMjacking and AI compute theft
- Agentic AI operations
- MCP security concerns
- Threat actor ecosystems
- Detection and mitigation strategies
